Для VS Code
Как использовать:
Распаковать архив
Открыть папку в VS Code, нажать F5

Для запуска нужно:
- MSYS2
- g++.exe по пути C:\msys64\ucrt64\bin\g++.exe
- gdb.exe по пути C:\msys64\ucrt64\bin\gdb.exe

Если gdb не установлен:
Скачать MSYS2 и в терминале выполнить:
pacman -S mingw-w64-ucrt-x86_64-gdb

Для корректной кириллицы:
выполнить в терминале chcp 65001
